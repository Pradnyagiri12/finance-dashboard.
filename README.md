# 💸 My Finance Dashboard

Welcome to the Finance Dashboard! This is a beginner-friendly project that helps you track your income, expenses, and total balance in a very simple way.

## 🌟 What Does It Do?

- **View Your Balance**: See your total money, income, and expenses right at the top.
- **Transaction History**: See a table of all the transactions you've made.
- **Two User Roles**: 
  - **Viewer**: Can only look at the data.
  - **Admin**: Can click the "Add Transaction" button to add a new record to the list. 

## 📂 Project Files

There are only three files in this project to keep things simple:

1. **`index.html`** - This is the main webpage. (Open this file to run the app!)
2. **`styles.css`** - This file makes the dashboard look beautiful with colors and dark theme.
3. **`app.js`** - This file holds all the logic to make the buttons work and handle the math using React.

## 🚀 How to Run the App (No Setup Required!)

You don't need to install anything complicated! Follow these easy steps:

1. Open the folder where these files are saved on your computer (`TEST1`).
2. Find the file named **`index.html`**.
3. **Double-click** it! 
4. The dashboard will instantly open in your default web browser (like Google Chrome or Microsoft Edge).

That's it! You can click the "Admin" button at the top right to try adding your own transactions.
